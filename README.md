# 카이로노미콘: 시간해방전

`index.html`을 브라우저에서 직접 열어 플레이하는 단일 실행형 탑다운 액션/디펜스 ARPG입니다. 별도 서버, 빌드 도구, npm 설치 없이 실행됩니다.

## 실행 방법

가장 안전한 방법:

1. 이 폴더를 엽니다.
2. `게임실행.bat` 또는 `게임실행.url`을 더블클릭합니다.
3. 브라우저에서 새 게임 또는 불러오기를 선택합니다.

직접 열기:

- `C:\Users\죠죠\Documents\Codex\2026-05-06\files-mentioned-by-the-user-hero\index.html`

주의: 브라우저 주소창에 `/C:/c/Users/...`처럼 입력하면 잘못된 경로라 열리지 않습니다. 올바른 형식은 `file:///C:/Users/.../index.html`입니다.

## 게임 흐름

- 카이로스 아카데미움에서 최초명령 코어를 지키는 6웨이브 방어전
- 포탑 최대 7개 설치 및 지속 업그레이드
- 학교 밖 8권역과 66수문장 토벌
- 레벨업/수문장 정화로 얻는 스킬 포인트를 ESC 메뉴의 Q/W/E/R 4개 트리에 직접 투자
- 패시브 시간각인 강화 선택
- 코어 습격 대응과 미니맵/보스 추적
- 최종보스 세이렌 모르 3페이즈 전투
- 최초명령 재작성 엔딩

## 조작

- 우클릭: 이동
- 방향키: 키보드 전용 모드 이동
- Q/W/E/R: 스킬
- Space: 상호작용 또는 회피/대시
- I: 인벤토리
- M: 미니맵 표시 전환
- Tab / Esc: 정보창과 메뉴
- F2: 마우스 중심 / 키보드 전용 조작 전환
- ESC 메뉴: 요약, 스킬, 설정, 장비, 목표, 기록 6개 탭으로 분리. `스킬` 탭의 별도 스킬 포인트 투자소에서 각 스킬 오른쪽 `+` 버튼으로 Q/W/E/R 트리를 직접 강화하고, `기록` 탭에서 클리어 스토리와 세계관을 확인
- 선택창: 마우스 클릭 또는 1/2/3/4, Q/W/E/R 단축키

## 저장

저장/불러오기는 브라우저 `localStorage`를 사용합니다. 브라우저 사이트 데이터 삭제, 다른 브라우저 사용, 시크릿 모드 사용 시 저장 데이터가 보이지 않을 수 있습니다.

## 2026-05-16 추가 정리

- 저장은 브라우저 `localStorage` 기반 3개 슬롯을 지원합니다. ESC 메뉴의 `설정 / 저장` 탭에서 슬롯 1~3을 선택한 뒤 저장/불러오기를 누릅니다.
- 슬롯 1은 기존 단일 저장 키를 그대로 사용하므로 이전 저장 데이터와 호환됩니다.
- 스킬 포인트는 ESC 메뉴의 `스킬` 탭에서만 투자합니다.
- Q/W/E/R 4개 스킬 트리 외에 남는 포인트를 `공용 스탯 각인`에 투자해 공격, 생명, 방어, 가속을 올릴 수 있습니다.
- 아르벨리아 합류 후 `아르벨리아의 유예` 패시브가 스킬 탭에 표시되며, 패시브도 같은 스킬 탭의 `+` 버튼으로만 강화합니다.
- 신규 패시브 `시간 파편 방벽`, `회수된 초침`을 추가해 렉이 적은 상시형 방어/쿨타임 보조 성장 루프를 보강했습니다.
- 일반/상급/대수문장의 체력, 피해, 크기 편차를 재조정했습니다. 일반 수문장은 너무 약하지 않게, 대수문장은 과도하게 튀지 않게 보정하고 상급 이상은 더 크게 보이도록 했습니다.
- W 시간가속 이펙트는 화면을 덮지 않도록 시각 반경을 제한했습니다.
- R 최초명령은 초반 위력을 조금 올리고, 풀강/보스 처형 비율은 낮춰 장기 밸런스를 완화했습니다.

현재 구조는 `index.html` 직접 실행을 기준으로 유지됩니다.

## 2026-05-16 모바일 조작 추가

- 설정의 `조작` 버튼은 이제 `마우스 중심 → 키보드 전용 → 터치 전용` 순서로 전환됩니다.
- 모바일/태블릿처럼 터치 포인터가 감지되는 환경에서는 기본적으로 `터치 전용` 조작이 우선 적용됩니다.
- 터치 전용 모드에서는 좌측 하단 조이스틱으로 이동하고, 우측 하단 Q/W/E/R 버튼으로 스킬을 사용합니다.
- `상호작용` 버튼은 Space와 같은 포탈 이동/귀환/회피 보조 행동을 수행합니다.
- 모바일 화면에서는 PC용 하단 스킬바를 숨기고, 미니맵과 HUD를 작게 줄여 전투 화면을 덜 가리게 했습니다.
- 조작 모드는 저장 데이터에 포함되며, `node tools\runtime-smoke-test.js`로 터치 모드 이동/저장 호환 검증을 수행했습니다.
